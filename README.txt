# TP-Web-Promo-WEB

Aplicación ASP.NET para la promoción de productos.  
Trabajo práctico para la materia **Programación 3** (UTN).

## Estructura del proyecto

- `Dominio/`: Modelos de datos
- `Negocio/`: Lógica de negocio
- `Presentacion/`: Interfaz web (Web Forms o MVC)

## Tecnologías utilizadas

- ASP.NET (.NET Framework)
- C#
- HTML / CSS / JavaScript
